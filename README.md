# Discord Music Bot

A simple Discord bot that plays audio from YouTube using yt-dlp.

## Features
- `!play <url>` — Play audio from YouTube
- `!stop` — Stop playback
- Auto-join voice channel

## Setup
1. `npm install`
2. Set `DISCORD_TOKEN` environment variable
3. `npm start`

## Requirements
- Node.js 18+
- yt-dlp installed
- ffmpeg installed
